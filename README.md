# Daily Air Quality Action Agent

## Problem Statement No. 15
AI-powered Daily Air Quality Action Agent built using LangFlow, IBM watsonx.ai and IBM Granite Models.

### Features
- AQI Data Retrieval (RAG)
- Personalized Health Recommendations
- Predictive Alerts
- Community Awareness Insights
- Multi-Agent Architecture
- Interactive Dashboard

### Agents
1. AQI Data Agent
2. Health Advisory Agent
3. Forecasting Agent
4. Community Action Agent
